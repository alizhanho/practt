# Python Practice Tasks

This repository contains a set of beginner Python practice tasks.  
All tasks are grouped by topic for easier learning and navigation.

## Topics

### 1. Date and Time
Tasks related to working with dates and time:
- print yesterday, today, and tomorrow
- subtract 5 days from the current date
- remove microseconds from `datetime`
- calculate the difference between two dates in seconds

### 2. Generators
Tasks related to creating and using generators:
- generator of squares up to `N`
- generator of squares from `a` to `b`
- generator of even numbers from `0` to `n`
- generator of numbers from `n` down to `0`
- generator of numbers divisible by `3` and `4`

### 3. Math
Tasks related to mathematical calculations:
- convert degrees to radians
- calculate the area of a trapezoid
- calculate the area of a parallelogram
- calculate the area of a regular polygon

### 4. JSON Parsing
Tasks related to working with JSON files:
- read data from a `.json` file
- extract required fields
- display data in a table format
- format output according to a given example

## Purpose
The purpose of this repository is to practice basic Python skills:
- using modules
- input and output
- functions
- generators
- calculations
- file handling
- JSON processing
